Each digit has a specific place and value. The number above is 478,235

The digit 4 is in the hundred thousands place, so its value is 400,000

Expanded Notation

Example: 8763

8763 = 8000 + 700 + 60 + 3

OR

(8 × 1000) + (7 × 100) + (6 × 10) + (3 × 1)

Expanded notation shows the value of each digit based on its place value position.

Lesson Summary

What we learned today:

  • Place value determines the value of each digit
  • Expanded form shows the value of each digit
  • Standard form is the normal way we write numbers
  • Numbers can be written in words
